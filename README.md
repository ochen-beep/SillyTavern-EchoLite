# EchoLite 
**EchoLite** — облегчённый форк [EchoChamber](https://github.com/mattjaybe/SillyTavern-EchoChamber) для SillyTavern.

(основная ветка теперь BETAjail, обновитесь, если еще не :)) )

Расширение генерирует отдельную фейковую чат-ленту в стиле Discord, которая реагирует на текущую сцену в чате. 

Если вам нужен именно большой мультитул с кучей встроенных стилей, livestream-режимом и более широкой экспериментальной поверхностью — тогда лучше смотреть в сторону оригинального [EchoChamber](https://github.com/mattjaybe/SillyTavern-EchoChamber).

---

## Что умеет эта версия?

- запоминает комментарии **по посту и по свайпу**;
- автоматически подгружает старые комментарии, если вы листаете чат назад;
- умеет работать в режиме **без сохранения**;
- позволяет править prompt-блоки для без ручного редактирования всего шаблона.

---

## Установка

Установите расширение через раздел **Extensions** в SillyTavern:

```text
https://github.com/ochen-beep/SillyTavern-EchoLite
```

После установки перезапустите или обновите SillyTavern.
